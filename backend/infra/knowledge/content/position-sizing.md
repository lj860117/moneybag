# 仓位管理：一只股/基不要超过多少

"把鸡蛋放在一个篮子里然后看好它"——这句话被很多人断章取义。集中持仓的代价，远比看起来大。

## 集中 vs 分散：数字说话

假设你的组合由 N 只独立等权资产构成，每只年化波动率为 30%（典型 A 股个股水平）：

| 持仓数量 | 组合年化波动率（估算） | 单只暴雷对组合影响 |
|---------|-------------------|------------------|
| 1 只    | 30%               | 归零 = -100%     |
| 5 只    | 约 13%             | 归零 = -20%      |
| 10 只   | 约 9.5%            | 归零 = -10%      |
| 20 只   | 约 7%              | 归零 = -5%       |
| 50 只   | 约 5%（接近市场）   | 归零 = -2%       |

**结论**：从 1 只增加到 10 只，波动率降低 68%；从 10 只到 50 只，再降低 47%。前期分散的边际效益远大于后期。

## 凯利公式的简化思维

凯利公式（Kelly Criterion）告诉我们，最优单笔押注比例为：

> **最优仓位 = 胜率 × 赔率 - 败率 / 赔率**

现实应用：即便你对一只股票有 60% 的胜率，凯利公式给出的最优仓位也很少超过 20%。大多数职业投资者使用"**半凯利**"（最优仓位的一半），以牺牲少量预期收益换取更大的安全边际。

## 实用仓位规则

| 规则 | 说明 |
|------|------|
| **单只个股 ≤ 10%** | 防止个股黑天鹅摧毁组合 |
| **单只基金 ≤ 20%** | 基金内部已有分散，可适当集中 |
| **单一行业 ≤ 30%** | 行业轮动风险不可忽视 |
| **单一国家/地区 ≤ 70%** | 保持一定全球化配置 |
| **所有高风险资产 ≤ 自身风险承受上限**| 取决于年龄和收入稳定性 |

## 强制分散的心理价值

人类天然倾向于"专注于自己最了解的"——这在工作上是优势，在投资上是陷阱：

- 你对公司 A 了解越多，越倾向于重仓
- 但了解一家公司并不等于能预测它的股价
- **信息优势在股价中已被定价**，散户的"研究"难以转化为超额收益

强制性的仓位上限，是防止过度自信最有效的制度性保护。

## 实操建议

1. 建立个人"仓位规则清单"，写下每种资产的上下限，投资前先查规则
2. 若某单只标的"忍不住"想超过 20%，往往是情绪信号而非理性分析
3. 宽基 ETF 可以例外：如单只沪深 300 ETF 占 40~50%，内部已分散 300 只股票
4. 仓位管理不是"一次设定"，需随市场价格变化定期再平衡
