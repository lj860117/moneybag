# 指数投资的逻辑

指数投资（Index Investing）是指买入跟踪市场指数的基金（如沪深 300 指数基金、标普 500 ETF），长期持有，不做选股和择时。它是"不战而胜"的投资策略。

## 核心论据

### 1. 大多数主动基金跑不赢指数

**SPIVA 2023 数据**（来源：S&P Global）：
- 美国：15 年期，88% 的大盘主动基金跑输标普 500
- 中国：5 年期，约 70% 的主动偏股基金跑输沪深 300（来源：Wind 2023 数据）

这不是因为基金经理"笨"——而是因为主动管理的费用（管理费 1.5% + 交易费用）在长期复利下形成巨大拖累。

### 2. 费率差距的长期影响

| 基金类型 | 典型年费率 | 100 万投 30 年（假设市场年化 10%）|
|---------|----------|-------------------------------|
| 指数基金（ETF）| 0.15% | 1,617 万 |
| 主动基金 | 1.50% | 1,006 万 |
| **差额** | **1.35%/年** | **611 万（少 38%）** |

每年看起来微不足道的 1.35% 费率差，30 年后吞噬了 38% 的终值。

### 3. "不战而胜"的数学

指数基金的回报 = 市场平均回报 - 极低费率 ≈ 市场平均回报

由于所有投资者加总 = 市场本身，扣除费用后：
- 主动投资者平均收益 = 市场回报 - 高费率 < 市场回报
- 被动投资者收益 = 市场回报 - 低费率 ≈ 市场回报

这是算术恒等式，不需要信仰。

## 指数投资的局限

1. **不能规避系统性风险**：熊市时指数一样跌。2008 年沪深 300 跌 65%。
2. **需要长期视角**：短期（1-3 年）指数可能大幅跑输某些主动基金。
3. **不适合追求超额收益**：指数投资的目标是"市场平均"，不是"最好的"。
4. **A 股尚有 alpha 空间**：中国市场散户占比高，机构优势仍存在。但随着机构化进程，alpha 在缩小。

## 实操建议

1. **核心仓位用指数基金**：70-80% 资产配置用宽基指数，降低费率。
2. **卫星仓位可配主动**：20-30% 可选少数优秀主动基金（但要能坚持 5 年以上）。
3. **全球分散**：沪深 300 + 标普 500 + 少量新兴市场，避免单一国家风险。
4. **定投优于择时**：指数基金 + 月定投 = 自动低买高卖（均值回归下）。
