# 定投策略的数学原理

定投（Dollar-Cost Averaging, DCA）是指在固定时间间隔（如每月）投入固定金额购买基金/股票的策略。它不是"最优策略"，但是"最适合普通人"的策略。

## 定投如何工作

每月投入 3000 元买沪深 300 指数基金：
- 基金净值高时：3000 元买到的份额少
- 基金净值低时：3000 元买到的份额多

结果：你的平均成本自动偏向"低价区"——这叫**成本平均效应**。

**数学公式**：定投的平均成本 = 总投入金额 ÷ 总购入份额 = 各期价格的**调和平均数**

调和平均数 ≤ 算术平均数（等号仅在所有价格相同时成立），因此定投成本总是 ≤ 简单平均价格。

## 定投 vs 一次性投入

**Vanguard 2012 研究结论**：

在美/英/澳三国市场 1926-2011 历史数据中：
- **一次性投入**在约 2/3 的时间段内跑赢定投（因为市场长期上涨，早投入 = 早享受复利）
- **定投**在剩余 1/3 时间段内跑赢（主要是下跌初期进入市场时）

结论：如果你有一笔闲钱且心理能承受，一次性投入期望收益更高。但如果你：
- 钱是按月赚来的（薪水）
- 心理上无法承受"买完就跌 30%"
- 没有择时能力

那么**定投是最适合你的策略**——它用"稍低的期望收益"换来了"显著降低的心理压力和行为风险"。

## 定投的真正价值

定投最大的价值不在数学，而在行为金融学：

1. **强制储蓄**：每月自动扣款，避免消费冲动吞噬投资资金。
2. **消除择时焦虑**：不需要判断"现在是不是高点"。
3. **降低后悔风险**：不会出现"all in 在最高点"的极端场景。
4. **培养纪律**：坚持 3 年以上的定投者，投资体验远优于频繁交易者。

## 定投的常见优化

### 1. 智慧定投（价值平均）

在基本定投基础上：
- 市场估值低时多投（如 PE < 历史 30% 分位时投 4000）
- 市场估值高时少投或暂停（如 PE > 历史 70% 分位时投 2000）

这叫"估值定投"或"智慧定投"，历史回测可提升年化 1-2%（来源：中证指数公司研究）。

### 2. 定投止盈

设定目标收益率（如累计 30% 或年化 15%），达到后赎回本金+部分利润，重新开始。适合震荡市（A 股常态）。

### 3. 定投标的选择

- **优先选波动大的宽基指数**：波动越大，定投的"低价多买"效应越显著。
- 沪深 300、中证 500 优于债券基金做定投标的。
- 债券基金波动小，定投和一次性投入差异不大。

## 常见误区

1. **"定投一定赚钱"** → 不一定。如果市场长期下跌且你在底部前赎回，一样亏。定投降低的是时点风险，不消除市场风险。
2. **"定投 3 个月没赚钱就放弃"** → 定投需要至少经历一个完整牛熊周期（A 股约 3-5 年）才能看到效果。
3. **"跌了就加倍投"** → 如果资金来源稳定可以适当多投，但不要超出承受范围。跌 50% 后再跌 50% 是可能的。
4. **"选最好的基金定投"** → 过去业绩最好的基金定投效果不一定好。选宽基指数更稳妥。

## 行动清单

1. 确定每月可定投金额（月收入的 20-30%）
2. 选择 1-3 只宽基指数基金
3. 设置银行自动扣款（每月固定日期）
4. 设定止盈目标（如年化 12% 或累计 30%）
5. 至少坚持 3 年，忽略短期波动
