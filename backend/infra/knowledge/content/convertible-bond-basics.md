# 可转债：介于股债之间的资产

可转债（Convertible Bond）是一种"债券 + 股票期权"的复合产品，享有"**进可攻、退可守**"的美誉——本质是债，在正股大涨时可转化为股票获利。

## 可转债的基本结构

- **面值**：通常 100 元/张，在二级市场可高于或低于面值交易
- **票息**：通常低于同级别纯债（1~2%），因为含有期权价值
- **转股价**：约定可按此价格将债券转为股票的价格
- **存续期**：通常 6 年

**进攻端**：若正股涨幅超过转股价 30%（即触发强赎线 130 元），持有人可转股获利；
**防守端**：若正股不涨甚至下跌，持有到期可收回本金+利息，损失有限。

## 中国 A 股可转债的特殊规则

| 规则 | 说明 |
|------|------|
| **强制赎回（强赎）** | 正股连续 15~20 个交易日收盘价 ≥ 转股价 × 130%，公司可按 103 元赎回 |
| **回售条款** | 正股连续低于转股价 70% 达一定天数，持有人可按 103 元回售给公司 |
| **下修转股价** | 公司可主动向下调整转股价，吸引投资者转股（常见于公司希望转股减少债务）|
| **税收** | 可转债利息收入按 20% 缴税；转股后的股票买卖按股票规则 |

**强赎是最重要的退出信号**：当公告强赎时，若不操作，最终只能拿到 103 元（而市价可能在 130+ 元）。

## 可转债的主要风险

1. **信用风险**：发行公司违约，债券无法兑付。历史上中国公募可转债鲜有违约，但小盘公司需警惕
2. **流动性风险**：部分小规模可转债交易量极低，买卖价差大
3. **正股不涨风险**：若正股长期横盘或下跌，可转债只能拿息，收益低
4. **强赎不操作的损失**：市价 135 元但强赎价 103 元，若忘记卖出直接损失 23%

## 如何投资可转债

| 方式 | 特点 |
|------|------|
| **单只可转债** | 需研究正股基本面，适合有一定分析能力的投资者 |
| **可转债基金** | 分散风险，省力，管理费约 0.6~1.2%/年 |
| **可转债指数基金** | 费率更低，被动跟踪可转债指数 |

**建议**：普通投资者可用 **5~10% 仓位**配置可转债基金，作为介于纯债和股票之间的过渡资产，在牛市中分享部分股票收益，在熊市中凭债性保护本金。
