## 一、苏格拉底式提问（AI 不给答案，帮你问对问题）

### 1.1 问题

金融小白最需要的不是答案，是**知道该问什么**。

### 1.2 改造

新增 `domain/services/advisor_questions_service.py`。在任何资产调整操作前，AI 先抛 3–5 个定制化问题：

- "这笔钱未来几年要用？"
- "如果明天跌 30%，你会不会失眠卖出？"
- "这笔钱放余额宝吃 2% 你能接受吗？如果不能，为什么？"
- "如果再买这只基金，你持仓集中度会到 X%，你知道这意味着什么吗？"
- "上次调仓距今 Y 天，上次调整效果如何？"

### 1.3 问题从哪来

根据用户当前操作 + 家庭画像 + 持仓状态，AI 从**问题模板库**（40–60 条）里挑 3–5 条组合。**不是 AI 自由发挥**。

**模板库结构**（`infra/knowledge/content/questions/`）：

```json
{
  "id": "q_use_horizon",
  "category": "time_horizon",
  "text": "这笔钱未来几年要用？",
  "applicable_when": ["any_buy"],
  "weight": 10
}
```

### 1.4 价值

行为金融学证明最有效的"防蠢"机制。让用户发现自己其实知道答案。

### 1.5 落地

```
infra/knowledge/content/questions/       # 40-60 条 JSON 问题模板
domain/services/advisor_questions_service.py
use_cases/generate_socratic_questions.py
```

## 二、三视角二次意见（结构化多角度）

### 2.1 问题

金融小白做重大决定，一个视角容易盲。

### 2.2 改造

重大调整前（金额占比 > 20% 或涉及大类配置），AI 以三视角各给一句 50–80 字评价：

| 视角 | 关注点 | 典型话术 |
|------|-------|---------|
| **保守派（格雷厄姆）** | 最坏情况 / 安全边际 | "假设这个行业跌 40%，你的现金流还能撑吗？" |
| **长期派（博格尔）** | 30 年尺度 | "30 年后回头看，这次调整比定投指数更好吗？" |
| **行为派（卡尼曼）** | 情绪 / 锚点 | "你是因为它真的便宜还是因为它跌了 30% 让你心动？" |

### 2.3 边界

三视角只**检视决策质量**，不预测市场、不给仓位、不推荐标的。从知识库固定话术模板里选，AI 做填空。

### 2.4 触发条件

| 条件 | 阈值 |
|------|------|
| 金额占总资产 | >20% |
| 涉及大类配置变化（如首次买黄金）| 是 |
| 单笔加仓让单资产突破 25% | 是 |

### 2.5 落地

```
infra/knowledge/content/multiviews/       # 话术模板（每视角 10-20 条）
domain/services/multi_view_advisor_service.py
use_cases/run_multi_view_review.py
```

## 三、基于持仓的每周金融小课（教育路径）

### 3.1 问题

用户永远"不懂金融"就永远依赖系统，健康关系是**越用越懂**。

### 3.2 改造

`scripts/weekly_education_cron.py`，每周推送一个 3 分钟小课：

- 持有基金 → 这周讲"ETF vs 主动基金的长期差别"
- 配了黄金 → 下周讲"黄金在组合里的对冲作用"
- 考虑提前还房贷 → 讲"提前还贷 vs 继续投资的数学"
- 看到浮亏 > 10% → 讲"为什么不该看浮亏而该看基本面"

内容全部来自 `08-knowledge-rag.md` 的知识库，AI 只做**"挑选 + 个性化改写"**。

### 3.3 目标

一年后用户能自己看懂报告，系统从"替你判断"变成"你的助手"。

### 3.4 推送策略

| 场景 | 推送时机 | 频率 |
|------|---------|------|
| 常规周课 | 每周日 20:00 | 1/周 |
| 持仓事件触发 | 单票浮亏 >10% 时 | 最多 1/月 |
| 新功能/知识库新增 | 有新文章时 | 不定 |

**推送疲劳控制**：周总上限 2 条（含事件触发），避免用户屏蔽。

### 3.5 落地

```
infra/knowledge/content/           # 知识库文章（同 08）
scripts/weekly_education_cron.py   # 每周日跑
use_cases/generate_weekly_lesson.py
```
