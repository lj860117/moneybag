STREAMING RESPONSE (SSE chunks):
  data: {"choices":[{"delta":{"content":"Hello"}}]}
  data: {"choices":[{"delta":{"content":" world"}}]}
  data: {"choices":[{"delta":{"reasoning_content":"Let me think..."}}]}  ← R1 only
  data: [DONE]

  ↓ Transformed to:

  data: {"delta":"Hello","source":"ai","done":false,"phase":"answering"}
  data: {"delta":" world","source":"ai","done":false,"phase":"answering"}
  data: {"delta":"Let me think...","source":"ai","done":false,"phase":"thinking"}
  data: {"delta":"","source":"ai","done":true}

To add new streaming patterns:
  1. Import: from services.llm_gateway import LLMGateway
  2. Get config: gw = LLMGateway.instance(); cfg = gw.get_api_config()
  3. Check quota: if not gw.pre_check(): return fallback
  4. Make request: async with httpx.AsyncClient() as client: ...
  5. Process SSE: async for line in resp.aiter_lines(): ...
  6. Check for reasoning_content (R1 models only)
  7. Emit with source tag and phase for state tracking

To add new multimodal patterns:
  1. Import: from services.llm_gateway import LLMGateway
  2. Get config: gw = LLMGateway.instance(); cfg = gw.get_api_config()
  3. Check quota: if not gw.pre_check(): return fallback
  4. Base64 encode: b64 = base64.b64encode(content).decode()
  5. Build messages with {"type": "image_url", "image_url": {"url": "data:...;base64,..."}}
  6. Make request: resp = await client.post(...)
  7. Parse: text = resp.json()["choices"][0]["message"]["content"]
  8. Extract JSON: json_match = re.search(r'\{[^}]+\}', text)
  9. Map fields to output schema
  10. Fallback to Tesseract if LLM fails
